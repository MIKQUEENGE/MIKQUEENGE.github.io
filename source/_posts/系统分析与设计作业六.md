# 用例文本编写

> 在大作业基础上，分析三种用例文本的优点和缺点

| 用例文本    | 优点               | 缺点               |
| ----------- | ------------------ | ------------------ |
| 详细用例    | 详细、深入         | 繁琐导致耗时长     |
| casual 用例 | 较详细、编写较简便 | 细节不足，不够正式 |
| brief用例   | 简介、编写简便     | 缺少细节           |
